EPPlus 2.7.0.1 

New features

* Comments 
* In-cell Richtext 
* Richtext added to drawing objects (Shape,Chart Title)
* Pagebreaks 
* A lot of new properties added to PrinterSettings like print area and print titles 
* Support for more advanced addresses like full row (1:1) full column (a:a) and multi (A1:A2,C1:C2) 
* Worksheet copy added to worksheets collection Add method (Workbook internal worksheets only)
* ..and alot of small fixes
